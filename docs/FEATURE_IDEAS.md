## 🎮 ゲーミフィケーション アイデア

| アイデア | 詳細 |
|---------|------|
| ストリークフリーズ | 月1回だけ「お休み券」でストリーク継続（有料 or バッジ報酬） |
| 習慣ごとのストリーク | 全体に加え「読書だけ7日」「運動だけ5日」等を個別表示 |
| 週間チャレンジ | 「今週5日連続で全習慣達成」等の期間限定ミッション |
| コンボ倍率 | 連続日数に応じてXPボーナス（例：7日連続で1.5倍） |
| レベルアップ演出 | レベルアップ時の演出・アニメーション強化 |
| パーフェクトウィーク | 週間全習慣達成で特別バッジ |

---

## 🌱 習慣づくり支援 アイデア

| アイデア | 詳細 |
|---------|------|
| 習慣テンプレート | 健康・仕事・学習・趣味等のカテゴリ別テンプレート |
| 小さく始める提案 | 「30分」→「5分から」等、小さなステップを提案 |
| 最適な時間の提案 | チェックイン傾向から「〇時ごろができそう」を提案 |
| 習慣のペアリング | 「運動の後にストレッチ」等の関連習慣を提案 |
| 初週ガイド | 新規習慣の1週目に習慣化のコツを tips 表示 |
| リマインダー | 習慣ごとに通知時間を設定（PWA プッシュ） |

---

## 📊 振り返り支援 アイデア

| アイデア | 詳細 |
|---------|------|
| 週次振り返り | 「今週どうだった？」のメモ＋感情記録 |
| インサイト表示 | 「水曜が達成率低い」「朝より夜が継続しやすい」等の傾向 |
| 月間サマリー | 月ごとの達成率・グラフ・一言コメント |
| 良かった点/改善点 | 週末に「うまくいったこと」「次に試したいこと」を記録 |
| 習慣の相関 | 「運動した日は読書もできた」等の傾向表示 |
| 気分ログ | 気分を記録し習慣との相関を見られるように |

## 📅 優先度の目安

1. **週次振り返り** - 振り返り習慣の核になる
2. **習慣テンプレート** - 習慣づくりのハードルを下げる
3. **習慣ごとのストリーク** - 既存ストリークを細分化
4. **インサイト表示** - データ蓄積後に実装しやすい
5. **リマインダー** - 継続に直結
