# Habitly - 習慣トラッカー

習慣を楽しく継続できる、モダンなトラッキングアプリです。

## 🎯 主な機能

### 習慣管理
- ✅ 習慣の作成・編集・削除
- 📅 週間グリッドで簡単チェック
- 📊 月間・年間の振り返り表示
- 🔄 週ナビゲーション（過去・未来の閲覧）

### ゲーミフィケーション
- 🎖️ 100種類のバッジ（実績）システム
- 👑 称号として装備可能
- ⭐ レベルシステム（XP獲得）
- 🔥 連続記録（ストリーク）

### データ管理
- 💾 localStorage による永続化
- 🔄 自動週更新
- 📱 レスポンシブデザイン

## 🚀 技術スタック

- **Framework**: Next.js 15 (App Router)
- **UI**: React 19 + TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Components**: shadcn/ui

## 📦 セットアップ

```bash
# 依存関係のインストール
pnpm install

# 開発サーバー起動
pnpm dev

# ビルド
pnpm build
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開きます。

## 📱 画面構成

### Home（ホーム）
- 週間グリッドでチェック管理
- 月・年単位の表示切り替え
- 装備中の称号表示
- XP/レベルバー

### Habits（習慣）
- 習慣の一覧表示
- 追加・編集・削除機能

### Profile（プロフィール）
- 獲得バッジ一覧
- 称号の装備/解除
- 統計情報（XP, レベル, ストリーク）

## 🎖️ バッジカテゴリ

- **初心者**: 最初の一歩、習慣コレクター
- **レベル**: Lv2, Lv5, Lv10, ... Lv100
- **連続記録**: 3日, 7日, 30日, 100日, 365日, 1000日
- **達成数**: 10回, 50回, 100回, ... 5000回
- **完璧な日**: 7日, 14日, 30日, 100日連続
- **特殊**: 早起き、夜型、カムバック、完璧主義者など

## 📝 主な機能実装

### 週の自動更新
新しい週が始まると自動的に今週に切り替わります。

### バッジ判定
チェックイン時に自動で条件を評価し、バッジを解除します。

### データ永続化
`localStorage` の `"habitly:v1"` キーに以下を保存:
- 習慣データ
- チェックイン履歴
- 週開始日
- 解除済みバッジ
- 装備中の称号

## 🔧 カスタマイズ

### バッジの追加
`lib/badges.ts` にバッジ定義を追加し、`evaluateBadges()` に判定ロジックを実装。

### テーマのカスタマイズ
`app/globals.css` でカラーパレットを変更可能。
